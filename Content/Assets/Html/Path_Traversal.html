Path Traversal (../)

Path traversal is a web security vulnerability that allows an attacker to read arbitrary files on the server that is running an application. Path traversal attacks take advantage of the way web servers handle files and directories. When a web server is asked for a file, it looks for the file in a specific directory. If the file is not found in that directory, the web server may look in other directories. This is called directory traversal. Path traversal attacks take advantage of this behavior to read arbitrary files on the web server. Path traversal attacks are possible when an application takes user input and uses that input to look for a file on the server. If the application does not properly validate the user input, an attacker may be able to manipulate the input to read arbitrary files on the server.
